Quick Start

<!-- Add to your HTML -->
<script src="https://your-domain.vercel.app/chatsi-analytics-sdk.min.js"></script>
<script>
  // Initialize SDK
  const analytics = new ChatsiAnalytics({
    posthogApiKey: 'YOUR_POSTHOG_KEY',
    posthogHost: 'https://us.i.posthog.com',
    platform: 'shopify', // or 'sfcc', 'magento', 'custom'
    debug: true
  });

  // Track events
  analytics.trackViewedProduct({
    product_id: '123',
    name: 'Product Name',
    price: 29.99,
    category: 'Electronics'
  });
</script>

Features

  • Universal platform support (Shopify, SFCC, Magento, Custom)
  • PostHog integration for advanced analytics
  • E-commerce event tracking (views, carts, purchases)
  • User identification and session tracking
  • Debug mode for development
  • CORS enabled for cross-origin requests
  • CDN-ready with aggressive caching
